# Система управления производством — ТОО KUS & ТОО SSAP

Веб-приложение для ежедневного сбора, отслеживания и анализа производственных показателей двух предприятий:

- **ТОО KUS** — производство электроэнергии и очистка воды
- **ТОО SSAP** — производство серной кислоты

## Возможности

- **Дашборд ввода данных** с переключением между предприятиями (KUS / SSAP), валидацией и автосохранением в `localStorage`
- **Последние показатели** с расчётом % выполнения плана и цветовой индикацией (зелёный ≥95% / оранжевый <95%)
- **История записей** (последние 10)
- **Общий отчёт и анализ**: фильтр по датам, итоговая статистика, линейный график выполнения плана, столбчатый график сравнения производства (двойная ось Y), таблица записей за период
- **Экспорт / импорт** данных в JSON (резервное копирование)
- Адаптивный дизайн (320–2560px), интерфейс на русском языке

## Технологии

- React 18.2 + Recharts 2.10 + Babel (через CDN)
- Хранилище — `localStorage` браузера
- Единый самодостаточный HTML-файл — без сборки

## Запуск

Откройте `production_app.html` в браузере (двойной клик) либо через опубликованную ссылку GitHub Pages.

Подробные требования — в [PRD.md](PRD.md).
